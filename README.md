# Syrah Centro Clínico — Website

Website institucional desenvolvido para o **Syrah Centro Clínico**, localizado em Águas Claras – Brasília/DF.
A plataforma apresenta a clínica, suas especialidades, equipe médica, convênios aceitos e facilita o contato e agendamento de consultas via WhatsApp.

O projeto foi pensado para transmitir **confiança, profissionalismo e excelência em saúde**, com foco especial em **nutrição, emagrecimento estratégico e bem-estar baseado em ciência**.

## Objetivos do projeto

* Criar uma presença digital profissional para a clínica
* Facilitar o agendamento de consultas
* Apresentar o corpo clínico e especialidades
* Informar convênios e formas de pagamento
* Oferecer uma experiência moderna, rápida e responsiva

## Funcionalidades

* Página institucional moderna
* Design responsivo (desktop, tablet e mobile)
* Sessão de especialidades médicas
* Apresentação do corpo clínico
* Lista de convênios aceitos
* Informações de pagamento
* Integração direta com WhatsApp para agendamento
* Otimização para SEO
* Interface elegante inspirada em clínicas premium
